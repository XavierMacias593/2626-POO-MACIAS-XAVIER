# Sistema de Restaurante - POO

Este proyecto es una implementación básica de un sistema de gestión de restaurante usando Programación Orientada a Objetos en Python.

## Estructura del proyecto

- `restaurante_app/main.py` - Punto de entrada del programa.
- `restaurante_app/modelos/producto.py` - Clase `Producto` que representa un plato, bebida u otro producto disponible.
- `restaurante_app/modelos/cliente.py` - Clase `Cliente` que representa un cliente del restaurante.
- `restaurante_app/servicios/restaurante.py` - Clase `Restaurante` que administra los productos y clientes.

## Ejecución

1. Abrir una terminal en `PARCIAL1/SEMANA4`.
2. Ejecutar:

```powershell
cd "PARCIAL1/SEMANA4/restaurante_app"
python main.py
```

## Funcionalidad demostrada

- Creación de objetos `Producto` y `Cliente`.
- Administración de productos y clientes desde la clase `Restaurante`.
- Visualización de información organizada en consola.
- Uso de importaciones entre módulos y paquetes.
- Uso de constructores `__init__()` y método especial `__str__()`.
